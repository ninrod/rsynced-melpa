A library which let's you set your custom indentation rules using a small EDSL.

Functions:

* indy

EDSL Functions:
* indy--next-tab
* indy--current-tab
* indy--prev-char
* indy--next-char
* indy--current-char
* indy--prev
* indy--next
* indy--current
* indy--ends-on
* indy--starts-with
* indy--contains
