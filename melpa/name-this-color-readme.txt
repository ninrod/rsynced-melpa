This package provides functions for getting a color's name based on hex RGB code.

Please see "README.org" for detailed instructions.
Documentation is also available at https://github.com/knl/name-this-color.el

Color names and shades are taken from 'Name that Color JavaScript' by Chirag
Mehta (http://chir.ag/projects/ntc/) and 'Color Name & Hue'
(http://www.color-blindness.com/color-name-hue/) by Colblindor.
