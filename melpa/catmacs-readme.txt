Provides a client for controlling Yaesu Transceivers using CAT protocol.
Initial capability supports FT-991(A)

Basic CAT functions work, including ...

   * Frequency
   * Mode
   * RF Gain
   * AF Gain
   * RF Attenuator
   * QMB Recall
   * Band Select
   * Band Up/Down
   * Encorder Up/Down
   * Channel Up/Down
   * Noise Blanker
   * LED/TFT Dimmer
   * and more ...
